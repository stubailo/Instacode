What is InstaCode?

InstaCode is a backend system and editor that lets you get set up to build web applications with Meteor. Even if you have never used Linux or a terminal before.

It comes with:

  1. One step installers for Windows and Mac
  2. A code editor called Brackets
  3. A Linux Virtual Machine so that you can enjoy the benefits of the world's most popular web server system while programming in your favorite OS
  4. Git integration for version control, code sharing, and downloading examples
  5. A toolbar of common Meteor commands so that you can make, run, and launch a real website with a few clicks
  6. Pre-packaged installers for many other languages such as Ruby and Python (coming soon!)

Why We Are Working On It

Programming for beginners comes with a steep learning curve.

This is because:

  1. Most online tutorials assume readers are using linux and are already familiar with common terminal commands. In reality, most beginners have Windows and have never used the terminal before.
  2. Setting up a programming environment is tedious and unintuitive. Downloading and installing all the necessary tools and software is discouraging to beginners.
  3. There is a significant disconnect between beginner coding tutorials and real life problems. Someone could have been learning Java for years and yet struggle to apply their skills on useful projects.

We started working on InstaCode at HackPrinceton 2014 to help remove these barriers to entry. Our goal is to make it so that everyone can get started programming without being discouraged by the setup process.
